Ember One is an open-source Bitcoin miner hashboard built using twelve Bitmain BM1362 ASIC chips, consuming roughly 100 Watts, and producing roughly 3.5Th/s. The RP2040 device-side firmware for the Ember One board management controller is licensed under the open-source GPLv3 license and the firmware source can be found [here](https://github.com/256foundation/emberone-usbserial-fw).
