A standalone MIDI interface for acoustic-style piano pedals compatible with Kawai GFP-3 triple pedal unit.

Supports USB MIDI 1.0/2.0, BLE MIDI and classic DIN-5 MIDI simultaneous broadcast. The project currently built around Pro Micro nRF52840 board with additional shielding and indication/control elements and a custom firmware.
