A swiss knife serial testing tool with USB interface to control a CAN/CAN-FD
bus interface, a full featured RS-232 interface and a RS-422/485 interface.
On top a RX/TX only RS-232 is equipped.
sertest-ng utilized a STM32G431RB microcontroler with a libopencm3 based
firmware. Suggestions are welcome.
