An open source [DMX512](https://en.wikipedia.org/wiki/DMX512) /
[RDM](https://en.wikipedia.org/wiki/RDM_(lighting)) transceiver.

The code can operate as either a DMX/RDM controller or a DMX/RDM receiver /
responder.
